| 维度 | 阈值 | 标注 |
|------|------|------|
| 价格涨跌 | ≥ +5% | 🚀 强势异动 |
| 价格涨跌 | +3% ~ +5% | 📈 温和上涨 |
| 价格涨跌 | -3% ~ -5% | 📉 温和下跌 |
| 价格涨跌 | ≤ -5% | 💥 大幅下跌 |
| 成交量 | ≥ 2x 日均量 | 🔥 成交量暴增 |
| 成交量 | 1.5x ~ 2x 日均量 | 📊 成交量偏大 |
| 52周高点距离 | ≤ 2% | 🏔️ 触碰历史高位 |
| 52周低点距离 | ≤ 2% | 🕳️ 触碰历史低位 |
